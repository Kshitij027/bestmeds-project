# BestMeds – Online Medicine Ordering System

BestMeds is a full-stack web application that allows users to browse medicines, view product categories, and place orders online.
The platform provides an easy way for customers to find health products and manage orders while giving administrators tools to manage products and categories.

---

## 🚀 Features

* User registration and login
* Browse medicines by categories
* Product search and filtering
* Add products to cart
* Order management system
* Admin panel for managing medicines and categories
* Secure backend APIs for data handling

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/Kshitij027/bestmeds-project.git
```

### 2. Navigate to the project

```
cd bestmeds-project
```

### 3. Setup Backend

```
cd backend
npm install
npm start
```

### 4. Setup Frontend

```
cd frontend
npm install
npm start
```

### 5. Setup Database

1. Open **MySQL**
2. Create a database named:

```
bestmeds
```

3. Import the file:

```
database.sql
```
